# GitHub Copilot Workflow: PDF → Master Markdown → Evidence-Only Research

## Prompt 1: Convert all PDFs into one master Markdown file

You are working inside this VS Code workspace.

Goal:
Convert all PDF files in this workspace folder into one consolidated Markdown file.

Input:
- The folder contains multiple PDF files.
- Only process `.pdf` files.
- Do not OCR images.
- Extract selectable PDF text only.
- If a PDF has no selectable text, record that clearly.

Output file:
- Create `master_input.md` in the workspace root.

Strict rules:
1. Process every PDF file found in the workspace.
2. Do not skip any PDF.
3. Sort files alphabetically by filename before processing.
4. Preserve document boundaries.
5. Preserve original filename.
6. Preserve page numbers if available.
7. Do not summarize.
8. Do not rewrite.
9. Do not infer missing text.
10. Do not fabricate citations.
11. Do not remove repeated wording.
12. Do not combine evidence across files.
13. If extraction fails for a file, add a section for that file and write the failure reason.

Required Markdown structure:

```md
